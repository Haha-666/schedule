# 🗓 排班表系統

純前端排班表網站，無需安裝任何後端或資料庫，直接開啟 `index.html` 即可使用。

## 功能

| 功能 | 說明 |
|------|------|
| 人員管理 | 新增 / 刪除員工，每人自動分配顏色 |
| 四種班別 | 早班(08–16)、中班(12–20)、晚班(16–00)、大夜班(00–08) |
| 休假設定 | 左側選取人員後，點日曆格子即可標記休假日 |
| 每班人數需求 | 設定每個班別最低需求人數，日曆顯示綠/黃/紅指示燈 |
| 自動排班 | 依需求人數與休假日自動分配，並平衡每人工作天數 |
| 手動調整 | 點擊日曆格子可逐人調整班別 |
| 班表總覽 | 底部表格顯示全月每人班次，含班次數與工時統計 |
| 匯出 CSV | 一鍵匯出 Excel 可開啟的 CSV 檔 |

## 使用步驟

1. **新增人員**：在左側「人員名單」輸入姓名，按「新增」
2. **設定需求**：在「每班需求人數」填入各班需要幾人
3. **設定休假**：點選左側人員 → 再點日曆中的日期標記休假
4. **自動排班**：按「⚡ 自動產生班表」
5. **手動調整**：直接點日曆格子，選擇人員再調整班別
6. **匯出**：按「📥 匯出 CSV」下載

## 執行方式

直接用瀏覽器開啟 `index.html` 即可，無需任何伺服器。

```bash
# 若想本地起一個 server
npx serve .
# 或
python3 -m http.server 8080
```
